## 🦆 DuckGame
A simple and fun guessing game where the player tries to guess how many rubber ducks are hiding.  
If the guess matches the actual number, the player wins.

### 🎮 Play the Game  
👉 [https://bblg520.github.io/duckGame.htm](https://bblg520.github.io/duckGame.htm)

### 🖼️ Screenshot
![DuckGame Screenshot](https://github.com/user-attachments/assets/a817e7f6-f77d-4c47-9111-7fedcd2aeaa5)

### 📌 Features
- Randomly generates a number of ducks between 2 and 10  
- Player inputs a guess  
- Displays win/lose result  
- Shows the actual number of ducks with images  

### 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
